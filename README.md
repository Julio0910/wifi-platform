\# Plataforma de Gestión WiFi

Proyecto para gestionar acceso WiFi en hoteles y centros de convenciones.



\## Estructura

\- apps/api  -> Backend (NestJS)

\- apps/web  -> Frontend (Next.js)

\- infra     -> Docker (Postgres/Redis)

\- db        -> SQL inicial (RLS)

\- prisma    -> Esquema/migraciones

\- docs      -> Documentación
